Goldcrest Defense

Goldcrest Defense is a 3D tower defense game created in Unity. This began as a Game Design course group project but I continued to work on it as a solo developer after the course ended. I worked as project manager and lead developer for game flow, bosses, animation, UI, and audio. Once converted to a solo project, I rewrote most aspects of the code base for optimization and readability purposes. The majority of the codebase is written in C#.

Features Overview

  • Four tower types each with four possible upgrades
  • Four enemy types (melee, archer, healer, soldier) with six tiers of difficulty
  • Four bosses, each equipped with unique abilities
  • Player abilities, unlocked after defeating bosses
  • Randomly spawning chests that provide cash or health
  • Local Trader. Sells upgrades to health, player speed, and an unlockable melee weapon
  • Animations for bosses, UI events, and tower attacks
  • Intricate wave spawner algorithm with various enemy sub-compositions
  • Tutorial with a dialogue tree which teaches the player aspects of the game while delivering lore
  • Zoomed Mode where the camera follows the player more closely
  • Audio for all attack actions, UI events, abilities, item collection, phase end, and more
  • Background music relevant to each game phase (Main Menu, Tutorial, Buy, Attack, Boss)
